R/V REX MARIS WEBSITE PACKAGE

Open index.html in any modern browser.

Files:
- index.html: website structure and copy
- styles.css: responsive visual design
- script.js: mobile navigation, reveal effects, gallery lightbox
- assets/images/: image assets extracted and cropped from the supplied slideshow PDF

To publish:
Upload the complete rex-maris-site folder to any static hosting service such as Netlify, GitHub Pages, Cloudflare Pages, or a standard web server.

Note:
The Google Fonts links require an internet connection. The site falls back to system fonts if they are unavailable.
